# InfoPacjent MVP

Frontend: React + Tailwind  
Backend: Node.js + Express  
Baza danych: Prisma + PostgreSQL

## Uruchomienie

### Backend

1. Przejdz do `backend`
2. Opcjonalnie skopiuj `.env.example` do `.env`
3. Uruchom `npm run dev`

API startuje domyslnie na `http://localhost:5000`.

### Frontend

1. Przejdz do `frontend`
2. Uruchom `npm run dev`

Frontend startuje domyslnie na `http://localhost:5173`.
